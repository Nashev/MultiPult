This folder contains demos showing how to perform some audio editing with NewAC components.
Following is the short description of each demo:

• AudioCutter - allows to select, play and save to disc a fragment of a wav file.

• AudioMixer - shows how to mix or concatenate two audio streams.

• FiltersTest – demonstrates properties of the windowed sinc and Chebyshev filters.

• MemoryInDemo - shows how to use TMemoryIn to generate a sine wave of a given frequency.

• RTMixer – demonstrates the use if the real-time mixer component.

• WaveFileConverter – this program can convert wav file parameters (the sample rate, the number of bits per sample, the number of channels).
